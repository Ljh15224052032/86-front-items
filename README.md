## 🚀 快速开始

### 本地开发
1. 克隆仓库
2. 使用 Live Server 打开 `projects/xx-project/index.html`
3. 开始编码

### 添加新项目
```bash
# 1. 在 projects/ 下创建新项目文件夹
mkdir projects/03-project-name

# 2. 编写项目文件

# 3. 提交代码
git add projects/03-project-name/
git commit -m "完成项目03：项目名称"
git push
```
