## 📝 如何调整

### 1️⃣ 调整发光位置

**位置**：`style.css` 第58-59行

```css
.beam-primary {
    top: 20%;    /* 垂直位置：0%(顶部) ~ 100%(底部) */
    left: 120%;  /* 水平位置：>100%(屏幕外) */
}
```

**位置示例**：
```css
/* 屏幕不同位置 */
top: 50%; left: 120%;   /* 右侧外，垂直居中 */
top: 20%; left: 110%;   /* 右侧外，靠上 */
top: 80%; left: 130%;   /* 右侧外，靠下 */

/* 左侧外 */
top: 50%; left: -10%;    /* 左侧外，垂直居中 */
top: 50%; left: -20%;    /* 左侧外，距离更远 */

/* 屏幕内 */
top: 20%; left: 80%;     /* 右上角 */
top: 50%; left: 50%;     /* 正中央 */
top: 80%; left: 20%;     /* 左下角 */
```

**💡 提示**：
- `left: 100%` = 屏幕右边缘
- `left: 120%` = 屏幕右边缘外20%
- `left: 50%` = 屏幕水平居中
- `left: 0%` = 屏幕左边缘

---

### 2️⃣ 调整摆动范围

**位置**：`style.css` 第101-107行

```css
@keyframes swing {
    from {
        transform: translate(-50%, -50%) rotate(150deg);  /* 起始角度 */
    }
    to {
        transform: translate(-50%, -50%) rotate(210deg);  /* 结束角度 */
    }
}
```

**调整示例**：

| 效果 | 起始角度 | 结束角度 | 范围 |
|------|---------|---------|------|
| 小范围（当前） | **150°** | **210°** | **60°** |
| 中等范围 | 140° | 220° | 80° |
| 大范围 | 120° | 240° | 120° |
| 只扫上半部分 | 150° | 180° | 30° |
| 只扫下半部分 | 180° | 210° | 30° |

**角度参考**：
- `0°` = 正上方
- `90°` = 正右方
- **180°** = **正左方**
- `270°` = 正下方

---

### 3️⃣ 调整光束强度

**位置**：`style.css` 第77-81行

```css
rgba(255, 230, 150, 0.1) 180deg,     /* 边缘 */
rgba(255, 240, 180, 0.2) 195deg,     /* 中等 */
rgba(255, 250, 200, 0.3) 210deg,     /* 核心 - 最亮 */
rgba(255, 240, 180, 0.2) 225deg,
rgba(255, 230, 150, 0.1) 240deg,
```

**强度等级**：

| 效果 | 边缘值 | 中等值 | 核心值 |
|------|--------|--------|--------|
| 很弱 | 0.05 | 0.1 | 0.15 |
| **弱（当前）** | **0.1** | **0.2** | **0.3** |
| 中等 | 0.15 | 0.25 | 0.4 |
| 强 | 0.2 | 0.35 | 0.5 |
| 很强 | 0.3 | 0.45 | 0.6 |

**💡 提示**：
- 核心值应该最大
- 三个值要平滑过渡（递增后递减）
- 推荐范围：0.1 - 0.5

---

### 4️⃣ 调整摆动速度

**位置**：`style.css` 第89行

```css
animation: swing var(--rotation-speed, 10s) ease-in-out infinite alternate;
```

**控制面板**：使用速度滑块（3-20秒）

**速度说明**：
- `3s`：很快（快速扫描）
- `10s`：中等（当前默认）
- `20s`：很慢（缓慢扫描）

## 📊 参数对照表

### 发光位置
| left值 | 位置说明 |
|--------|---------|
| <0% | 屏幕左侧外 |
| 0% | 屏幕左边缘 |
| 50% | 屏幕水平居中 |
| 100% | 屏幕右边缘 |
| **>100%** | **屏幕右侧外** |

### 角度参考
| 角度 | 方向 |
|------|------|
| 0° | 正上方 |
| 45° | 右上方 |
| 90° | 正右方 |
| 135° | 右下方 |
| **180°** | **正左方** |
| 225° | 左下方 |
| 270° | 正下方 |
| 315° | 左上方 |

### 透明度（亮度）
| 值 | 效果 |
|-----|------|
| 0.05 | 几乎看不见 |
| 0.1 | 很弱 |
| **0.2-0.3** | **适中（推荐）** |
| 0.4-0.5 | 明亮 |
| 0.6+ | 过曝（不推荐） |
