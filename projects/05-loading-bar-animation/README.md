# 项目05：Loading Bar Animation

## 📝 项目说明
这是一个动态加载条波浪动画效果项目，通过多个竖条的缩放和透明度变化，形成波浪式的加载动画。

## 🎯 实现功能
- 7个竖向加载条
- 波浪式动画效果（通过animation-delay实现）
- 缩放、位移、透明度同步变化
- 阴影效果增强立体感

## 🔧 技术要点
- CSS Flexbox 弹性布局
- CSS Keyframes 关键帧动画
- CSS transform（translateX、scale）
- CSS animation 属性（delay、duration、timing-function、iteration-count）
- CSS nth-child 选择器
- CSS box-shadow 阴影效果
