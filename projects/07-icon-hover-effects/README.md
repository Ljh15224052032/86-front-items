# 项目07：Icon Hover Effects

## 📝 项目说明
这是一个带有缩放动画效果的图标悬停项目。鼠标悬停时，圆形背景从中心向外扩散，同时图标颜色从深灰色变为白色。

## 🎯 实现功能
- 4个圆形图标按钮（礼物、酒杯、地球、毕业帽）
- 悬停时背景从中心向外缩放展开
- 悬停时图标颜色从深灰变为白色
- 平滑的过渡动画效果
- 使用 Font Awesome 4.7.0 图标库

## 🔧 技术要点
- CSS 伪元素（:before）实现动画层
- CSS transform（scale）实现缩放动画
- CSS transition 实现平滑过渡
- CSS border-radius 实现圆形
- CSS z-index 图层管理
- position 定位（absolute、relative）
- float 浮动布局（横向排列）
- line-height 实现垂直居中
- text-align 实现水平居中
