# 项目08：Transparent Text Effects

## 📝 项目说明
这是一个使用 CSS 混合模式实现的透明文字效果项目。文字与背景图片产生叠加效果，创造出独特的视觉体验。

## 🎯 实现功能
- 全屏背景图片
- 超大字体文字（10em）
- 使用 mix-blend-mode 实现文字与背景的混合效果
- 文字阴影和半透明背景
- 文字垂直居中显示

## 🔧 技术要点
- CSS mix-blend-mode 混合模式
- CSS text-shadow 文字阴影
- CSS background 背景图片设置
- CSS transform 实现垂直居中
- CSS position 绝对定位
- vh 视口高度单位
- em 相对单位（基于字体大小）
- rgba 半透明颜色
