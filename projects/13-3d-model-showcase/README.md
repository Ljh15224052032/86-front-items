# 项目13：3D Model Showcase

## 📝 项目说明
这是一个工业设计作品集的3D模型展示页面，用于展示 Simworks / 溪流 等工业设计模型。

## 🎯 实现功能
- ✅ 左右分栏布局（左侧60%为3D模型，右侧40%为作品信息）
- ✅ Sketchfab iframe嵌入3D模型
- ✅ 渐变色背景，现代化设计风格
- ✅ 响应式设计，适配桌面/平板/手机
- ✅ 详细的作品信息展示（设计参数、亮点、作者信息）
- ✅ 自定义滚动条样式

## 🏗️ 技术架构
- 使用 **Sketchfab** 平台托管3D模型
- 通过 **iframe** 嵌入到网页
- **Flexbox** 布局实现左右分栏
- **CSS渐变** 背景美化页面
- **媒体查询** 实现响应式适配

## 📋 页面布局
| 区域 | 内容 |
|------|------|
| 左边 (60%) | Sketchfab 嵌入的3D模型（可交互） |
| 右边 (40%) | 作品名称、设计说明、渲染图、尺寸/材质参数 |

## 🎨 设计特点
- 渐变紫色背景 (#667eea → #764ba2)
- 白色卡片容器，圆角阴影
- 左侧黑色背景突出3D模型
- 右侧信息层次分明，易于阅读

## 📱 响应式断点
- **≥ 1200px（桌面）**：左右布局，60%:40%
- **≤ 1199px（平板）**：上下布局，模型50vh高度
- **≤ 768px（平板竖屏）**：优化间距和字体
- **≤ 576px（手机）**：进一步缩小模型区域至35vh

## 📌 踩坑记录

### 问题1：Sketchfab模型导出
**解决方案**：
- 在SOLIDWORKS中选择"另存为"
- 导出格式优先选择：OBJ > STL > STEP
- 本项目使用的是OBJ格式导出的模型

### 问题2：Sketchfab嵌入代码
**注意事项**：
- iframe需要设置 `width: 100%` 和 `height: 100%` 才能填充容器
- 必须保留 `allowfullscreen` 属性以支持全屏查看
- 模型加载需要几秒钟，网速慢时可能需要等待

### 问题3：Flexbox布局
**关键点**：
- 使用 `flex: 0 0 60%` 固定左侧宽度
- 使用 `flex: 0 0 40%` 固定右侧宽度
- 容器需要设置 `overflow: hidden` 防止内容溢出圆角

### 问题4：响应式设计
**策略**：
- 平板及以下改为上下布局（`flex-direction: column`）
- 模型区域使用vh单位（50vh、40vh、35vh）确保高度
- 信息区添加 `overflow-y: auto` 支持滚动
