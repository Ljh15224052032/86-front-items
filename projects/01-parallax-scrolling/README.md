# 项目01：视差滚动效果

## 📝 项目说明
这是一个视差滚动效果的示例项目。

## 🎯 实现功能
- 多层背景以不同速度滚动
- 鼠标滚动时触发视差动画
- 流畅的过渡效果

## 🔧 技术要点
- CSS3 `transform: translateZ()`
- JavaScript 监听滚动事件
- `perspective` 创建3D空间
