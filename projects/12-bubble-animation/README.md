# 项目12：Bubble Background Animation

## 📝 项目说明
这是一个气泡背景动画项目。在红色背景上，10个不同大小的半透明白色气泡从底部不断上升并旋转，创造出梦幻般的视觉效果。

## 🎯 实现功能
- 10个不同尺寸的气泡（15px - 150px）
- 气泡从底部上升并旋转（720度）
- 气泡在上升过程中从方形变为圆形
- 气泡逐渐变透明直至消失
- 每个气泡有不同的位置、速度和延迟
- 无滚动条全屏展示
- 左上角说明文字框

## 🔧 技术要点
- CSS Keyframes 关键帧动画
- CSS transform（translateY、rotate）
- CSS opacity 透明度过渡
- CSS border-radius 圆角过渡
- CSS animation-delay 动画延迟
- CSS animation-duration 动画时长
- CSS nth-child 选择器
- vh 视口高度单位
- rgba 半透明颜色
