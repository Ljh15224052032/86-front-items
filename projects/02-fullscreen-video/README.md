# 项目02：全屏视频背景

## 📝 项目说明
这是一个全屏视频背景的示例项目。
除了第一个，后续的注释就不会太多了，只会对一些比较重要的功能进行注释

## 🎯 实现功能
- 视频作为全屏背景
- 视频循环播放
- 内容层叠加在视频之上
- 响应式布局

## 🔧 技术要点
- HTML5 Video 标签
- CSS object-fit 属性
- z-index 层级控制
- 响应式设计
