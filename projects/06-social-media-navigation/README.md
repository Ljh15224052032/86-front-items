# 项目06：Social Media Navigation with Hover Effects

## 📝 项目说明
这是一个带有悬停动画效果的社交媒体导航栏项目。鼠标悬停时，背景色会从右向左展开，产生流畅的视觉过渡效果。

## 🎯 实现功能
- 5个社交媒体按钮（Facebook、Twitter、Google+、LinkedIn、Instagram）
- 悬停时背景色从右向左展开动画
- 每个按钮有对应的品牌颜色
- Font Awesome 图标集成
- 导航栏居中显示

## 🔧 技术要点
- CSS 伪元素（:before）实现动画层
- CSS transform（scaleX）实现缩放动画
- CSS transform-origin 控制缩放起点
- CSS transition 实现平滑过渡
- CSS nth-child 选择器为不同按钮设置样式
- CSS 伪类（:hover、:last-child）
- Flexbox 弹性布局
- 绝对定位 + transform 居中
- z-index 图层管理
- Font Awesome 图标库
